# Stack

## What are stack?
A stack is a linear data structure that stores items in a "Last in, first out"/"First in,Last out"

## How to use:

```python
# creating.
stack = []
 
# append() function to push
# element in the stack
stack.append('a')
stack.append('b')
stack.append('c')

print(stack)

# pop() function to pop
# element from stack in
# LIFO order
print(stack.pop())
print(stack.pop())
print(stack.pop())

print(stack)
```
